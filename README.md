# Fitness Buddy - PS13

AI-powered conversational fitness coach using IBM watsonx Orchestrate and Granite LLM.

# Features as per PS-13
1. Recommend home workouts and routines based on user input
2. Provide motivational tips and daily fitness inspiration  
3. Suggest simple, nutritious meal ideas
4. Encourage habit-building and consistency

# Disclaimer
General fitness and basic nutrition guidance only. Not medical advice.

# Technology
IBM watsonx Orchestrate + IBM Cloud Lite
